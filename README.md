# Git Workflow Simulation

Repository ini digunakan untuk praktikum simulasi workflow Git dan GitHub.

## Tujuan

Praktikum ini berfokus pada:
- Fork repository
- Branching
- Commit bertahap
- Push ke fork
- Pull Request
- Sinkronisasi dengan upstream
- Penyelesaian merge conflict

## Struktur Project

Project ini terdiri dari:

- index.php
- style.css
- script.js

## Catatan

Repository ini tidak berfokus pada pengembangan aplikasi.
Fokus utama adalah praktik penggunaan Git secara profesional.
